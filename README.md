## 🌍 Overview
**DICPMS** is a comprehensive platform that empowers communities and government agencies by providing:

- **A mobile app for citizens:** Track infrastructure projects, report property impact, and follow compensation requests.
- **Government dashboards:** Monitor ongoing projects, manage compensation workflows, and generate real-time analytics.

The system bridges the gap between citizens and authorities, ensuring **transparency, accountability, and efficiency** in public infrastructure management.

---

## 🎯 Objectives
- ✅ Allow citizens to report property impacts via mobile app
- ✅ Digitally track compensation requests
- ✅ Provide real-time dashboards for government agencies
- ✅ Reduce disputes and improve transparency
- ✅ Maintain a secure, auditable record of all project activities

---

## ⚠️ Problem Statement
Infrastructure projects often lead to:

- Property damage or acquisition
- Delays in compensation
- Poor data management
- Lack of transparency

**DICPMS solves this** by providing a unified digital solution for both citizens and government authorities.

## ✨ Key Features

<details>

### 🏠 Mobile App for Citizens
- Report damaged or affected properties
- Track compensation request status
- Receive notifications on project updates
- Submit supporting documents/photos
- Receive notifications in case their property will be affected by the project 

### 📊 Government Dashboard
- Monitor project progress daily
- Track property compensation workflows
- Generate analytics and reports
- Role-based access for officers, valuers, auditors
### Government Design platform 
- Design projects digitally
- Get instant details on the people's properties which will be affected
  

### 🔐 Security
- Role-based access control
- Encrypted data storage
- Audit logging for all transactions

### 💰 Compensation Management
- Automated valuation
- Payment tracking
- Dispute resolution management
